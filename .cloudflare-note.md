# Cloudflare Pages Configuration

このプロジェクトはCloudflare Pagesのデフォルト規約を使用しています。

## 必要な設定（Cloudflare Dashboard）

### D1 Database Binding
- Variable name: `DB`
- D1 database: `fit-analyzer-db`
- Database ID: `d63d86b2-a801-46ef-9839-a14c8eaf053a`

### Workers AI Binding  
- Variable name: `AI`

### Environment Variables (Production)
- `APP_URL`: `https://fit-analyzer.pages.dev`
- `GITHUB_CLIENT_ID`: (your GitHub OAuth client ID)
- `GITHUB_CLIENT_SECRET`: (your GitHub OAuth client secret)
- `GOOGLE_CLIENT_ID`: (your Google OAuth client ID)
- `GOOGLE_CLIENT_SECRET`: (your Google OAuth client secret)

## Functions

Functions are located in `dist/functions/` after build:
- `/api/auth/register` -> `dist/functions/api/auth/register.js`
- `/api/auth/login` -> `dist/functions/api/auth/login.js`
- `/api/auth/me` -> `dist/functions/api/auth/me.js`
- `/api/auth/logout` -> `dist/functions/api/auth/logout.js`
- `/api/auth/github` -> `dist/functions/api/auth/github.js`
- `/api/auth/google` -> `dist/functions/api/auth/google.js`
- `/api/analyze` -> `dist/functions/api/analyze.js`
- `/api/workouts` -> `dist/functions/api/workouts.js`
